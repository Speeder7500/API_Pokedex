# Pokedex API

## Prérequis
Cloner le dépôt à l'endroit ou vous voulez.
Assurer vous d'avoir node d'installer

## Lancement
Ouvrer un invite de commande à l'endroit du fichier index.js et taper la commande suivante 'node index.js'.
L'API est démarée, vous pouvez l'utiliser en allant sur un navigateur internet et tapez '<votre adresse ip>:5001' pour attérir sur la page d'acceuil.

## Fonctionnement

Vous pouvez rechercher un pokémon spécifique par son id ou son nom.
Vous pouvez afficher l'entièreté des pokémon présents dans le pokédex.
Vous pouvez faire des combats avec des pokémons sélectionés aléatoirement dans le pokedex.
Vous pouvez également avoir un pokémon sélectioné au hasard dans le pokédex
